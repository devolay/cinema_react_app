# Cinemasio

Showcase Project - React, TypeScript, Redux with Firebase (Api service & athentication) and Material-UI with styled-components.

## Demo

Project is available at [Link](https://cinemasio.web.app/)

## Start

```
yarn
yarn start
```

or

```
npm i
npm run start
```
